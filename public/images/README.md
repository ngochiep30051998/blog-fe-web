# Blog Images

This directory should contain the blog post images referenced in the blog pages.

## Required Images

Based on the current blog setup, you need the following images:

- `blog-1.jpg` - For "Getting Started with Next.js 14 App Router"
- `blog-2.jpg` - For "Modern CSS Techniques for 2024"
- `blog-3.jpg` - For "TypeScript Best Practices for React"
- `blog-4.jpg` - For "Building Responsive Layouts with Tailwind CSS"
- `blog-5.jpg` - For "Optimizing Performance in Next.js Applications"
- `blog-6.jpg` - For "The Future of Web Development"

## Image Specifications

- **Format**: JPG or PNG
- **Recommended size**: 1200x675px (16:9 aspect ratio)
- **File size**: Optimize images to be under 500KB for better performance
- **Naming**: Use descriptive names like `blog-1.jpg`, `blog-2.jpg`, etc.

## How to Add Images

1. Export images from Figma design
2. Optimize images using tools like:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - Next.js Image Optimization (automatic)
3. Place optimized images in this directory
4. Update image paths in blog post data if needed

## Note

If you're exporting from Figma, make sure to:
- Export at 2x resolution for retina displays
- Use appropriate format (JPG for photos, PNG for graphics with transparency)
- Maintain consistent aspect ratios across all blog images
